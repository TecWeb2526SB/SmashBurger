# Documentazione Progetto SmashBurger
Progetto di Tecnologie Web 2026.

## Specifiche Architetturali
Il progetto segue il pattern Model-View-Controller (MVC) semplificato:
- `index.php`: Agisce come controller principale per la homepage.
- `includes/`: Contiene la logica di backend e le configurazioni centralizzate.
- `views/`: Contiene il markup HTML dinamico e i frammenti (template).
- `styles/`: Risorse frontend (CSS, JS, Immagini) organizzate in modo modulare.
